# Various OMICS projects

# (1) DNA Translator
DNA Translator - a simple tool for translating amino acids sequence into RNA, DNA codons sequence (back-translation)

# Usage

Set a parameter: 
	PDB Id - Protein Id RCSB Protein Data Bank,
	eg. 3e7y for Insulin

# Example
	PDB: 3e7y
	AA1: GIVEQCCTSICSLYQLENYCNFVNQHLCGSHLVEALYLVCGERGFFYTPK
	RNA DEFAULT: GGUAUUGUUGAGCAGUGUUGUACUUCUAUUUGUUCUUUGUAUCAGUUGGAGAAUUAUUGUAAUUUUGUUAAUCAGCAUUUGUGUGGUUCUCAUUUGGUUGAGGCUUUGUAUUUGGUUUGUGGUGAGCGUGGUUUUUUUUAUACUCCUAAG
	DNA DEFAULT: GGTATTGTTGAGCAGTGTTGTACTTCTATTTGTTCTTTGTATCAGTTGGAGAATTATTGTAATTTTGTTAATCAGCATTTGTGTGGTTCTCATTTGGTTGAGGCTTTGTATTTGGTTTGTGGTGAGCGTGGTTTTTTTTATACTCCTAAG
	DNA ALL: 
	['GGT', 'GGC', 'GGA', 'GGG']
	['ATT', 'ATC', 'ATA']
	['GTT', 'GTC', 'GTA', 'GTG']
	['GAA', 'GAG']
	['CAA', 'CAG']
	...
